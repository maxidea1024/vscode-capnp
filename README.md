# Capnproto support for vscode

Support for capnproto (http://capnproto.org/) 

## Features
- Syntax Highlighting
- Bracket matching
- Capnp ID generator ( Ctrl+p -> GenerateCapnpID )


Built specially for @grimpy

## Installation
clone repo under `~/.vscode/extensions/vscode-capnp`.
